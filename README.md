# Advent of Code

My task solving on the Golang

## Installation

- Install the latest version of Go https://go.dev/dl/ 
- Go to the https://adventofcode.com/2022 and login
- Take inputs, and paste them to `/inpits/input-**.txt`
- Or use my predefined inputs

## Usage

```
go run task-01.go
```
