# Advent of Code

Foobar is a Python library for dealing with word pluralization.

## Installation

- Install latest version of Go https://go.dev/dl/ 
- Go to the https://adventofcode.com/2022 and login
- Take inputs, and paste them to `/inpits/input-**.txt`

## Usage

```
go run task-01.go
```
